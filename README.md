## 🔍 Component Overview
The Environmental and Safety Monitoring System is a system that helps keep people safe during fun run events. When people are running, a sensor called DHT11 checks the temperature and humidity all the time. Then ESP32 microcontroller looks at the information it gets. Sends it to the Blynk Cloud using Wi-Fi. The Environmental and Safety Monitoring System then checks the information it gets against some rules that say what is safe and what is not. It shows what is happening in time on a special screen. If the Environmental and Safety Monitoring System sees that it is hot or too humid it sends a warning message to the people in charge so they can do something about it right away and help in case of an emergency. The Environmental and Safety Monitoring System is very helpful, for the people who organize fun run events.

## ⚡Quick Start Guide
1. Connect the DHT11 sensor to the ESP32 according to the circuit design.
2. Power on the ESP32 using a USB cable or portable power bank.
3. Ensure the ESP32 is connected to the configured Wi-Fi network.
4. Upload and run the monitoring program using Arduino IDE.
5. Open the Blynk mobile application or web dashboard.
6. Verify that temperature and humidity readings are displayed correctly.
7. Monitor the dashboard for real-time environmental data and automated safety alerts.
8. Begin the fun run event with continuous environmental and safety monitoring active.
