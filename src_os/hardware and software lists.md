# Smart Fun Run-Team A:Environmental and safety Monitoring

***

### 💾 Hardware Requirements

* **ESP32 Development Board** – Main IoT microcontroller for data collection and Wi-Fi communication.
* **DHT11 Temperature & Humidity Sensor** – Measures environmental temperature and humidity.
* **USB Cable** – Used for programming and powering the ESP32.
* **Laptop** – Runs the Python monitoring server and stores data logs.
* **Wi-Fi Network/Hotspot** – Enables communication between ESP32, Blynk Cloud, and the monitoring server.
  
  ***

### 💻 Software Requirements

* **Arduino IDE** – Used to develop and upload code to the ESP32.
* **Python 3.x** – Runs the monitoring server and database management scripts.
* **Blynk IoT Platform** – Provides real-time dashboard visualization and remote monitoring.
* **OpenPyXL Library** – Handles Excel file creation and data logging.
* **WiFi Library (`WiFi.h`)** – Enables ESP32 wireless connectivity.
* **Blynk Library (`BlynkSimpleEsp32.h`)** – Connects the ESP32 to the Blynk platform.
* **DHT Sensor Library (`DHT.h`)** – Interfaces with the DHT11 sensor.
* **Threading Module (`threading`)** – Supports concurrent processing and mutex synchronization in the Python server.
* **Socket Module (`socket`)** – Enables network communication between IoT devices and the server.

  ***

### 🔧Development Environment

* **Operating System:** Windows 10/11, macOS, or Linux
* **Programming Languages:** C++ (ESP32 Firmware) and  Python (Backend Server & Database Management)
